# Landing Page - Advocacia Premium

Uma Landing Page profissional, sofisticada e de alta conversão desenvolvida especificamente para advogados autônomos e escritórios de advocacia de alto padrão.

## Visão Geral
Este projeto foi desenvolvido com foco em performance, acessibilidade e conversão, utilizando uma paleta de cores elegantes (Dark Navy e Dourado) e tipografia moderna para transmitir máxima credibilidade e autoridade. O site consiste em uma aplicação "Single Page" (Página Única), totalmente responsiva para funcionar perfeitamente em Smartphones, Tablets e Desktops.

## Tecnologias e Recursos
* **HTML5 Semântico**: Estruturação moderna visando boas práticas de SEO.
* **CSS3 Vanilla**: Estilização total sem dependência de frameworks externos pesados. Uso de variáveis CSS (`:root`) para cores, estilos flexíveis (Flexbox/Grid), e Glassmorphism.
* **JavaScript Vanilla**: Lógica leve responsável pelas Animações de Scroll (Intersection Observer), Acordeão de FAQ, rolagem suave de menu e Menu *Hamburger* para mobile.
* **RemixIcon**: Biblioteca de ícones via CDN (HTTPS).
* **Google Fonts**: Fontes 'Playfair Display' para os títulos (trazendo autoridade) e 'Inter' para os textos.

## Hospedagem Fácil com GitHub Pages

Como o site é estático (apenas HTML, CSS e JS), **ele é perfeitamente e 100% nativo para hospedagem gratuita utilizando o GitHub Pages**.

Para publicá-lo:
1. Crie um repositório no seu GitHub.
2. Suba todos os arquivos (`index.html`, `style.css`, `script.js` e a pasta `assets/`).
3. Nas configurações do repositório (**Settings**), vá para a seção **Pages** (no menu da esquerda).
4. Em **Build and deployment**, selecione **Deploy from a branch**.
5. Selecione a branch `main` e a pasta `/(root)`.
6. Conclua clicando em **Save**. Em instantes, o GitHub fornecerá o link público do seu site!

## Customização
* **Cores**: Para alterar a identidade visual, basta modificar as variáveis localizadas na linha 2 do arquivo `style.css` (no seletor `:root`).
* **Imagens**: Para trocar as fotos, basta substituir os arquivos na pasta `assets` ou alterar o caminho (`src`) diretamente no `index.html`.
* **WhatsApp**: Para alterar o número do WhatsApp, procure no arquivo `index.html` pelas tags de link com `href="https://wa.me/..."` e troque o número.
